¿Por qué usar HTML en Moodle?

Moodle es una herramienta potente, pero su diseño estándar suele ser lineal y estático. Al integrar HTML, usted rompe esa barrera, permitiendo:

Contenidos en tarjetas y banners.
Elementos interactivos y desplegables.
Diseños adaptables a móviles.
Mayor enganche del estudiante.

Usted enseña, la IA diseña

No necesita ser un programador experto. La Inteligencia Artificial actúa como su **traductor técnico**. Usted solo describe su idea pedagógica y la IA genera el código HTML necesario para Moodle.

01

Describa

Pida un banner, un menú o una grilla de recursos.

02

Copie

Tome el código HTML generado por la herramienta.

03

Pegue

Use el editor de código en Moodle y ¡listo!

Cómo pedir ayuda a la IA

Use estos "prompts" (instrucciones) para obtener resultados profesionales.

Petición Semántica

Sea específico con los colores y la estructura para que la IA no improvise.

Moodle Friendly

Pida que el CSS esté "en línea" (inline) para evitar conflictos con el tema de Moodle.

Prompt para ChatGPT / Claude
// Ejemplo de prompt para un banner de bienvenida "Actúa como un experto en diseño instruccional y HTML5. Genera un código HTML con estilos en línea (inline CSS) para un banner de bienvenida en Moodle de color turquesa. El banner debe tener un título, un párrafo corto motivacional y un botón centrado que diga 'Comenzar Unidad'. Asegúrate de que sea totalmente responsive y que use una tipografía moderna como Montserrat." // La IA te devolverá un código que solo deberás copiar.

Potencial Ilimitado

Grillas de Recursos

Organice lecturas, videos y tareas en cuadros elegantes en lugar de una lista larga de archivos.

Acordeones de Información

Oculte textos extensos que el estudiante puede desplegar al hacer clic, ahorrando espacio visual.

Alertas y Notas

Cree cuadros de 'Importante' o 'Dato Curioso' que resalten visualmente sobre el resto del texto.
