En la Nube

Aprenda a publicar sus desarrollos HTML en Netlify y GitHub para integrarlos como recursos interactivos de alta gama en sus cursos de Moodle.

¿Por qué no subir el archivo directo a Moodle?

Aunque Moodle permite subir archivos, usar repositorios externos ofrece ventajas críticas para la ingeniería y el diseño:

Velocidad: Los recursos cargan mucho más rápido desde servidores dedicados.
Actualización: Si corrige el código en el repositorio, se actualiza automáticamente en Moodle.
Interactividad: Permite usar JavaScript complejo que a veces Moodle bloquea por seguridad.

El Flujo de Trabajo

01. Crea con IA
02. Publica en Nube
03. Enlaza a Moodle

Opción A: Netlify Drop

La ruta más rápida para docentes. Sin instalaciones.

Netlify Drop es como un "buzón" mágico. No necesita saber programar ni usar comandos. Solo necesita su carpeta con el archivo HTML y arrastrarla a una ventana de su navegador.

app.netlify.com/drop

Arrastre su carpeta aquí

Paso a paso con Netlify

01

Preparación

Guarde su desarrollo como un archivo llamado index.html y métalo en una carpeta nueva en su escritorio.

02

Publicación

Vaya a Netlify Drop y arrastre esa carpeta. ¡En 3 segundos tendrá un enlace público!

03

Enlace Final

Copie la URL que Netlify genera (ej: mi-curso.netlify.app). Ese es el link que usaremos en Moodle.

Opción B: GitHub Pages

Ideal para repositorios institucionales y control de versiones.

GitHub es la biblioteca de código más grande del mundo. Usar GitHub Pages le permite tener una dirección personalizada del tipo usuario.github.io/proyecto.

Ventaja Académica:

Permite que otros docentes colaboren en el recurso o que usted mantenga un historial de cambios para cada semestre.

Es como tener un servidor propio y gratuito con el respaldo de Microsoft.

Paso a paso con GitHub

01

Nuevo Repo

En su cuenta de GitHub, cree un "Repository" nuevo y suba su archivo HTML (botón 'Upload files').

02

Activar Pages

Vaya a Settings > Pages. En la sección 'Branch', seleccione 'Main' y haga clic en Save.

03

Listo para usar

Espere un minuto y GitHub le dará un link verde. Su recurso ya es una página web oficial.

El Toque Final: Integrar en Moodle

¿Cómo hacer que el link de Netlify o GitHub se vea dentro del aula?

Uso de IFrames

Un "Iframe" es una ventana que muestra un sitio web dentro de otro. Moodle permite insertarlos fácilmente en una **Etiqueta** o una **Página**.

Código para Moodle:

<iframe src="URL_DE_NETLIFY" width="100%" height="600px" frameborder="0"></iframe>

En el editor de Moodle, busque el icono </> y pegue el código.
